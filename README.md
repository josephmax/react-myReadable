how to launch?
---
1. download the repository with `git clone "https://github.com/josephmax/react-myReadable.git"`
2. clone the server code with `git clone "https://github.com/josephmax/my_readable_server_starter.git"`
3. `cd ~your directory/my_readable_server_starter/`
4. run `yarn` or `npm i`
5. run `node server`
6. `cd ~your directory/react-myReadable`
7. run `yarn` or `npm i`
8. run `yarn start` or `npm run start`
